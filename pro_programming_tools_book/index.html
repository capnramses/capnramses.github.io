Professional Programming Tools for C and C++

A book by Anton Gerdelan and Katja Žibrek

About This Book

This book presents a quick start to a full range of tools you can use for programming and shipping quality software written in the C or C++ programming languages. Each chapter addresses an important program development task, and introduces tools for completing the task on all the major desktop operating systems. We try to minimise the discussion, and get you started right away with practical instructions, adding hints and tips for common issues at the end of the chapter.

Programming books are mostly written for entry-level introduction to topics or at a very advanced level and there isn't much bridging material available for those in the years in-between. When we started writing this book we were thinking of coders in the experience ranges between doing a degree and more senior career programming roles. In particular, coders wanting to get into work in the industry, but unsure if they are missing some skills, tools, and techniques that weren't part of an academic education or learned for research projects. Our reviewers have been programming for many years, and they picked up some useful new tools and techniques from reading this text, so there may well be something new here for you too.

C and C++ programming languages are very powerful, but have some memory and security vulnerabilities that require special attention. This book will get the reader up to speed with standard programming tools used in many languages, but also with specialist tools like fuzzers and static code analysers that are particularly useful for finding flaws in C and C++ programs, and may not have been learned in a university course or on the job.
