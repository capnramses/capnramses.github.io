Anton's WebGL Notes

I'm starting to work with WebGL for my new job at Trinity College. I haven't used Javascript since about 1960 but I'm well into 3D programming, so my approach is going to be to slowly build up bits and pieces that replicate the more useful bits of my OpenGL 4 tutorials, rather than skip steps using someone else's pre-canned wrapper. This way I can learn the points of difference between WebGL and OpenGL with C. I'll post things that I make/learn/find here. This will be something like a research journal but less formal (read messy).
